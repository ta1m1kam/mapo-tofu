# 麻婆豆腐増殖

麻婆豆腐の豆腐がどんどん増えていくサイト。

## Demo

https://ta1m1kam.github.io/mapo-tofu/

## Stack

- Bun + Vite + SolidJS + TypeScript + UnoCSS
- ポップ用フォント: **Mochiy Pop One** + **M PLUS Rounded 1c** (Google Fonts)
- GitHub Actions → GitHub Pages 自動デプロイ

## Features

- 380ms ごとに豆腐がポンッと増える
- 画面タップで +1
- 一定数で「💥 100こ達成！」のお祝いテキスト
- 「⏸ 止める / ▶ もっと / 🌶 最初から」ボタン
- 唐辛子が浮遊、キラキラ ✨ がポップ

## Develop

```sh
bun install
bun run dev
```
